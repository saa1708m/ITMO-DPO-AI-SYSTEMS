# Экзамен / зачет

Раздел предназначен для материалов промежуточной аттестации по дисциплине.

## Возможное содержание

- перечень вопросов;
- кейсовые задания;
- практико-ориентированные задачи;
- критерии оценивания ответа;
- шкала перевода баллов в итоговую оценку.

## Назначение

Экзамен или зачет подтверждает достижение ключевых результатов обучения по компетенциям LC-2, LC-3 и LC-5 и позволяет оценить целостное понимание студентом жизненного цикла промышленной ML-системы.
