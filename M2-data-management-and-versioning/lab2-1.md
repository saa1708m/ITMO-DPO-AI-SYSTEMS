## Назначение

Сформировать практические навыки проведения разведочного анализа данных, очистки данных и визуализации с использованием Python.

## Проверяемые результаты обучения

Студент должен уметь:

- загружать и анализировать данные в Pandas;
- исследовать структуру данных, пропуски, распределения и выбросы;
- строить визуализации для анализа данных;
- выполнять базовую подготовку данных;
- формулировать выводы и предложения по дальнейшей подготовке данных.

## Постановка задания

1. Выберите открытый датасет, содержащий числовые и, желательно, категориальные признаки.
2. Загрузите данные в Pandas DataFrame и выполните начальный просмотр: первые строки, типы данных, размерность.
3. Проведите описательную статистику по числовым признакам.
4. Исследуйте пропуски, предложите и реализуйте способ их обработки.
5. Выявите выбросы с помощью box plot или метода IQR и обоснуйте способ работы с ними.
6. Постройте визуализации:
   - гистограммы для числовых признаков;
   - box plot для числовых признаков;
   - корреляционную матрицу;
   - графики по категориальным признакам при их наличии.
7. Выполните преобразования:
   - кодирование категориальных признаков;
   - при необходимости нормализацию или стандартизацию;
   - создание хотя бы одного нового признака.
8. Сформулируйте выводы по структуре данных, качеству данных и следующим шагам подготовки.
9. Оформите результат в виде Jupyter Notebook с кодом, визуализациями и пояснениями.

## Формат сдачи

Обязательные артефакты:

- при необходимости `data/processed/` или отдельный CSV/Parquet с подготовленными данными.

Ссылка на Git-репозиторий предоставляется преподавателю.
