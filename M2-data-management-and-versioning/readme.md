# Модуль 2. Управление данными и их версионирование

## Назначение модуля

Модуль охватывает управление данными, контроль качества данных, версионирование наборов данных и базовые практики воспроизводимости исследований.

## Проверяемые компетенции

- LC-2
- Индикаторы: LC-2.1, LC-2.3

## Состав материалов

-  — стартовый шаблон КИМ для модуля;
-  — вложения, схемы, дополнительные материалы.

## Комментарий по заполнению

В этот раздел далее следует добавить конкретные лабораторные работы, тесты, кейсы, рубрики оценивания и при необходимости ключи или эталонные материалы.
