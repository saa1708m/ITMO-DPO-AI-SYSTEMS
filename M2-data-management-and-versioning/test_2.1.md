# Тест 2.1. EDA и data quality

## Паспорт КИМ

| Параметр | Значение |
|----------|----------|
| **Вид КИМ** | Тест |
| **Модуль** | 2. Управление данными и их версионирование |
| **Тема** | EDA и data quality для ML |
| **Компетенция** | LC-2 |
| **Индикатор** | LC-2.1 |
| **Уровень** | Б (базовый) |
| **Время выполнения** | 30 минут |
| **Максимальный балл** | 10 |

## Назначение

Проверить теоретические знания о сборе, первичном анализе, очистке и оценке качества данных для задач машинного обучения.

## Проверяемые результаты обучения

Студент должен:

- знать основные этапы EDA;
- понимать методы очистки данных: обработку пропусков, выбросов и дубликатов;
- знать метрики качества данных;
- иметь представление о роли Great Expectations в обеспечении качества данных.

## Инструкция студенту

Вам предлагается 10 вопросов с выбором одного правильного ответа. За каждый правильный ответ начисляется 1 балл. Максимальный балл — 10.

---

### Вопрос 1

Что такое EDA (Exploratory Data Analysis)?

A) Процесс очистки данных от выбросов  
B) Первичный анализ данных для понимания их структуры, распределений и взаимосвязей  
C) Метод машинного обучения для прогнозирования  
D) Процесс преобразования данных в нужный формат  
E) Способ визуализации данных в виде дашбордов
