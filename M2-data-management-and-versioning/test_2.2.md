# Тест 2.2. Версионирование данных и feature-пайплайны

## Паспорт КИМ

| Параметр | Значение |
|----------|----------|
| **Вид КИМ** | Тест |
| **Модуль** | 2. Управление данными и их версионирование |
| **Тема** | Версионирование данных и feature-пайплайны (Git LFS, DVC, ETL/ELT, feature store, автоматические проверки качества) |
| **Компетенция** | LC-2, LC-5 |
| **Индикатор** | LC-2.3, LC-5.2 |
| **Уровень** | С (средний) |
| **Время выполнения** | 30 минут |
| **Максимальный балл** | 10 |

## Назначение

Проверить знания о версионировании данных, построении ETL/ELT-пайплайнов, использовании feature store и автоматизации проверок качества данных в ML-проектах.

## Проверяемые результаты обучения

Студент должен:

- знать различия между Git LFS и DVC и понимать их назначение;
- понимать концепции ETL и ELT;
- иметь представление о feature store и его роли в ML-пайплайнах;
- знать инструменты для автоматической проверки качества данных.

## Инструкция студенту

Вам предлагается 10 вопросов с выбором одного правильного ответа. За каждый правильный ответ начисляется 1 балл. Максимальный балл — 10.
