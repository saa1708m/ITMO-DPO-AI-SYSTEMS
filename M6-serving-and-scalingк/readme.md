# Модуль 6. Развёртывание и масштабирование моделей

## Назначение модуля

Модуль посвящён развёртыванию моделей как сервисов, архитектурным решениям для serving и базовым стратегиям масштабирования.

## Проверяемые компетенции

- LC-3, LC-5
- Индикаторы: LC-3.2, LC-5.3

## Состав материалов

-  — стартовый шаблон КИМ для модуля;
-  — вложения, схемы, дополнительные материалы.
