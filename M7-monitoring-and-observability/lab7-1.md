# Лабораторная работа 7.1. Настройка мониторинга ML-сервиса

## Паспорт КИМ

| Параметр | Значение |
|---|---|
| Вид КИМ | Лабораторная работа |
| Модуль | 7 |
| Тема | Наблюдаемость и метрики ML-систем |
| Компетенция | LC-5 |
| Индикатор | LC-5.4 |
| Уровень | Б |
| Режим выполнения | Индивидуально / в паре |
| Время выполнения | 1–2 занятия |
| Максимальный балл | 10 |

## Назначение

КИМ предназначен для практической проверки умения настраивать базовый мониторинг ML-сервиса и интерпретировать ключевые сигналы наблюдаемости.

## Проверяемые результаты обучения

- выделяет ключевые инфраструктурные, сервисные и прикладные метрики;
- настраивает сбор и визуализацию метрик;
- задаёт базовые правила алертинга;
- формулирует выводы по данным мониторинга.

## Постановка задания

Необходимо организовать базовый мониторинг ML-сервиса, развёрнутого локально или в контейнерной среде. Требуется:

1. выбрать сервис или учебный прототип;
2. определить набор метрик для базовой наблюдаемости;
3. настроить сбор и визуализацию метрик;
4. подготовить дашборд;
5. задать не менее одного алерта;
6. сформулировать вывод о состоянии сервиса.

## Формат сдачи

В состав сдаваемых материалов должны входить:

- краткое описание наблюдаемой системы;
- перечень выбранных метрик;
- конфигурация мониторинга или её фрагменты;
- скриншоты / экспорт дашборда;
- описание правила алертинга;
- итоговый вывод.

## Критерии оценивания

| Критерий | Баллы |
|---|---:|
| Выбор и обоснование метрик | 0–2 |
| Корректность настройки мониторинга | 0–2 |
| Качество дашборда и визуализации | 0–2 |
| Настройка алертов | 0–2 |
| Выводы и оформление | 0–2 |
| **Итого** | **0–10** |

## Шкала перевода баллов

| Баллы | Оценка |
|---:|---|
| 9–10 | «Отлично» |
| 7–8 | «Хорошо» |
| 5–6 | «Удовлетворительно» |
| 0–4 | «Неудовлетворительно» |

## Методические примечания

Рекомендуется по возможности разделять панели наблюдаемости по уровням: инфраструктура, сервис, прикладные сигналы. Предпочтение следует отдавать не количеству метрик, а их эксплуатационной значимости.
