# Лабораторная работа 7.2. Drift-мониторинг и алерты

## Паспорт КИМ

| Параметр | Значение |
|---|---|
| Вид КИМ | Лабораторная работа |
| Модуль | 7 |
| Тема | Дрейф данных и моделей |
| Компетенция | LC-5 |
| Индикатор | LC-5.4 |
| Уровень | С |
| Режим выполнения | Индивидуально / в паре |
| Время выполнения | 1–2 занятия |
| Максимальный балл | 10 |

## Назначение

КИМ предназначен для практической проверки умения организовывать drift-мониторинг, интерпретировать сигналы деградации и проектировать базовые действия по реагированию.

## Проверяемые результаты обучения

- организует сравнение текущих и референсных данных;
- выявляет признаки drift;
- интерпретирует drift-сигналы с точки зрения сопровождения модели;
- предлагает правила алертинга и действия по реагированию.

## Постановка задания

Необходимо реализовать учебный сценарий drift-мониторинга для модели или датасета. Требуется:

1. выбрать референсный и текущий набор данных;
2. определить контролируемые признаки;
3. сравнить распределения;
4. зафиксировать обнаруженные изменения;
5. предложить правила алертинга;
6. сделать вывод о необходимости дальнейших действий.

## Формат сдачи

В состав сдаваемых материалов должны входить:

- описание данных и сценария;
- перечень контролируемых признаков;
- результаты сравнения распределений;
- описание drift-сигналов;
- проект правил алертинга;
- итоговый вывод.

## Критерии оценивания

| Критерий | Баллы |
|---|---:|
| Выбор данных и признаков | 0–2 |
| Корректность drift-анализа | 0–2 |
| Качество интерпретации сигналов | 0–2 |
| Обоснованность алертов и реагирования | 0–2 |
| Оформление и воспроизводимость | 0–2 |
| **Итого** | **0–10** |

## Шкала перевода баллов

| Баллы | Оценка |
|---:|---|
| 9–10 | «Отлично» |
| 7–8 | «Хорошо» |
| 5–6 | «Удовлетворительно» |
| 0–4 | «Неудовлетворительно» |

## Методические примечания

При анализе drift-сигналов следует различать статистическое отклонение и эксплуатационно значимую деградацию. В выводах важно указывать не только факт drift, но и предполагаемое последующее действие.
