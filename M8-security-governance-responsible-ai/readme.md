# Модуль 8. Безопасность, governance и ответственный ИИ

## Назначение модуля

Модуль охватывает управление секретами, базовые практики RBAC, model cards, аудит и вопросы responsible AI в промышленной эксплуатации ML-систем.

- LC-5
- Индикаторы: LC-5.5

## Состав материалов

-  — стартовый шаблон КИМ для модуля;
-  — вложения, схемы, дополнительные материалы.
