# Проектная работа

Проектная работа является интегральным оценочным средством по дисциплине и предназначена для проверки того, как обучающийся связывает данные, моделирование, архитектурные решения и инженерную эксплуатацию ML-системы.

## Что должно быть отражено

- постановка задачи;
- данные и подготовка данных;
- обучение и сравнение моделей;
- контейнеризация и воспроизводимость;
- CI/CD или элементы автоматизации;
- развёртывание или serving;
- мониторинг, безопасность, документация.

## Сдаваемые материалы

- ссылка на репозиторий проекта;
- отчёт или техническая записка;
- артефакты модели и экспериментов;
- материалы для демонстрации и защиты.
