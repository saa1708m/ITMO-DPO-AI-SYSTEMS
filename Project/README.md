# Курсовой проект по дисциплине «Промышленная разработка, развёртывание и сопровождение ML-систем»

## Общая информация

| Параметр | Значение |
|----------|----------|
| **Дисциплина** | Промышленная разработка, развёртывание и сопровождение ML-систем |
| **Направление** | 01.03.02 Прикладная математика и информатика |
| **Уровень** | Бакалавриат |
| **Курс** | 3–4 (в зависимости от учебного плана) |
| **Семестр** | 6–7 |
| **Объём** | 32 а.ч. (самостоятельная работа) |
| **Форма контроля** | Защита курсового проекта |
| **Максимальный балл** | 10 |

## Цель курсового проекта

Спроектировать, разработать и внедрить сквозное ML-решение, охватывающее полный жизненный цикл системы искусственного интеллекта: от анализа данных до промышленной эксплуатации с мониторингом и обеспечением безопасности.

## Структура проекта

Проект выполняется в формате сквозного end-to-end ML-решения и включает следующие этапы:

| Этап | Содержание | Результат |
|------|------------|-----------|
| 1 | Анализ, очистка и версионирование данных | DVC-репозиторий, EDA-отчёт |
| 2 | Эксперименты, модели, MLflow | Эксперименты, Model Registry |
| 3 | Контейнеризация | Dockerfile, docker-compose |
| 4 | CI/CD и оркестрация | GitHub Actions, Airflow DAG |
| 5 | Развёртывание и масштабирование | REST API, Kubernetes |
| 6 | Мониторинг | Prometheus/Grafana, Evidently |
| 7 | Безопасность и Governance | Vault, RBAC, model card |

## Перечень тем

Полный список из **10 тем** представлен в файле [topics.md](topics.md).

Краткий перечень:
1. Разработка интеллектуального ИИ-ассистента для обработки обращений клиентов
2. Система интеллектуального анализа и проверки документов
3. MLOps-платформа для мониторинга промышленного оборудования
4. ИИ-агент для автоматизации закупочных процессов
5. RAG-система для работы с корпоративной документацией
6. Система рекомендаций для подбора персонала
7. Сервис прогнозирования спроса и оптимизации ассортимента
8. Голосовой ИИ-агент для медицинских учреждений
9. Система видеоаналитики для безопасности на производстве
10. Платформа для генерации и поиска синтетических данных
