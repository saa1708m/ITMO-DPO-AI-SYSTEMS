# Кейс №6. Система рекомендаций для подбора персонала

## Паспорт КИМ

| Параметр | Значение |
|----------|----------|
| **Вид КИМ** | Курсовой проект (кейс) |
| **Модули** | 1, 2, 3, 5, 6, 7 |
| **Компетенции** | LC-2, LC-3, LC-5 |
| **Индикаторы** | LC-2.1–2.3, LC-3.1, LC-5.1–5.4 |
| **Уровень** | С (средний) |
| **Режим выполнения** | Индивидуально |
| **Срок выполнения** | 3 недели |
| **Максимальный балл** | 10 |

## Назначение

Разработать HR-рекомендательную систему на основе данных о сотрудниках. «Сайбокс.Персонал» от Т1 аккумулирует и анализирует данные о работе сотрудников, позволяя HR-специалистам быстрее формировать кадровый резерв и управлять им.

## Постановка задания

### 1. Подготовка данных (Модуль 2)
- Соберите данные о сотрудниках (навыки, проекты, оценки, опыт).
- Проведите EDA, очистите данные.

### 2. Разработка моделей (Модуль 3)
- Реализуйте модель для рекомендации сотрудников на проект.
- Используйте как минимум 2 подхода: коллаборативная фильтрация, градиентный бустинг.

### 3. CI/CD и оркестрация (Модуль 5)
- Настройте CI/CD для деплоя.

### 4. Развёртывание API (Модуль 6)
- Создайте REST API для получения рекомендаций.

### 5. Мониторинг (Модуль 7)
- Настройте мониторинг качества рекомендаций (precision@k, recall@k).
