## Назначение

Разработать систему прогнозирования спроса на товары и оптимизации ассортимента на основе исторических данных продаж. Решение востребовано в ритейле для повышения эффективности управления запасами.

## Постановка задания

### 1. Подготовка данных (Модуль 2)
- Используйте открытый датасет (например, Walmart Sales Forecasting).
- Проведите EDA временных рядов.

### 2. Разработка моделей (Модуль 3)
- Реализуйте модели прогнозирования спроса: Prophet, XGBoost, LSTM.
- Сравните модели в MLflow.

### 3. CI/CD и оркестрация (Модуль 5)
- Настройте Airflow DAG для автоматического переобучения.

### 4. Развёртывание API (Модуль 6)
- Создайте API для получения прогнозов.

### 5. Мониторинг (Модуль 7)
- Настройте мониторинг ошибок прогноза (MAPE drift).
