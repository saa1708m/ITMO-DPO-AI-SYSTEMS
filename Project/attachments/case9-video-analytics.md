# Кейс №9. Система видеоаналитики для безопасности на производстве

## Паспорт КИМ

| Параметр | Значение |
|----------|----------|
| **Вид КИМ** | Курсовой проект (кейс) |
| **Модули** | 1, 2, 4, 5, 6, 7 |
| **Компетенции** | LC-2, LC-3, LC-5 |
| **Индикаторы** | LC-2.1, LC-3.1, LC-5.1–5.4 |
| **Уровень** | С (средний) |
| **Режим выполнения** | Команда 2–3 человека |
| **Срок выполнения** | 3 недели |
| **Максимальный балл** | 10 |

## Назначение

Разработать систему компьютерного зрения для мониторинга безопасности на промышленных объектах. Применение «Сайбокс» для видеоаналитики.

## Постановка задания

### 1. Подготовка данных (Модуль 2)
- Используйте открытый датасет для детекции объектов (например, COCO).
- Выполните разметку или выборку данных.

### 2. Разработка моделей (Модуль 3)
- Реализуйте модель детекции объектов (YOLOv8 или Faster R-CNN).
- Обучите модель на выбранных данных.

### 3. Контейнеризация (Модуль 4)
- Создайте Dockerfile для сервиса обработки видео.

### 4. CI/CD и оркестрация (Модуль 5)
- Настройте CI/CD для деплоя.

### 5. Развёртывание и мониторинг (Модули 6–7)
- Разверните API для обработки видео.
- Настройте мониторинг производительности (FPS, latency).

3. Как вы организовали масштабирование обработки нескольких видеопотоков?
4. Как вы настроили мониторинг производительности системы?
5. Какие дополнительные сценарии безопасности можно реализовать на основе видеоаналитики?
