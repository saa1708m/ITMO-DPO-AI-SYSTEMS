# Методические указания

Раздел содержит материалы для студентов и преподавателей, связанные с выполнением заданий, оцениванием, использованием ресурсов и организацией контроля.

- `students/` — рекомендации обучающимся;
- `teachers-assessment/` — рекомендации преподавателям по оцениванию;
- `teachers-resources/` — рекомендации преподавателям по использованию ресурсов.
