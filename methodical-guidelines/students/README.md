# Рекомендации обучающимся

Студентам рекомендуется:

- заранее знакомиться с целями и проверяемыми результатами каждого модуля;
- вести аккуратную историю изменений в репозитории и документировать решения;
- сохранять промежуточные артефакты экспериментов, обучения, контейнеризации и развёртывания;
- при выполнении лабораторных работ соблюдать требования к воспроизводимости и оформлению отчётов;
- при использовании генеративного ИИ фиксировать, какие части работы выполнены самостоятельно, а какие — с использованием внешних инструментов.
