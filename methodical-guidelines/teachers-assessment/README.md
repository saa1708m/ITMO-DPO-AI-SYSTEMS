# Рекомендации преподавателям по оцениванию

При оценивании рекомендуется учитывать:

- достижение заявленных результатов обучения по каждому модулю;
- воспроизводимость решения;
- корректность технической реализации;
- качество инженерной документации и аргументации;
- понимание студентом собственных решений на защите или при устном обсуждении.

Для лабораторных работ и проектной работы желательно использовать рубрики с уровнями освоения: базовый, средний, продвинутый.
