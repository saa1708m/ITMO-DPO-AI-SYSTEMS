# Рекомендации преподавателям по использованию ресурсов

В разделе `resources/` рекомендуется поддерживать структурированный набор учебных и инженерных материалов:

- литература и статьи;
- датасеты;
- библиотеки и инструменты;
- шаблоны промптов;
- тестовые и задачные банки.

При обновлении ресурсов желательно указывать, в каком модуле и для какого КИМа они используются.
