## Требования к добавлению

Для каждого нового датасета рекомендуется указывать:

- источник (локальный, открытый, партнёрский) и ссылку, если доступен внешний ресурс;
- лицензию или условия использования;
- состав признаков и целевую переменную;
- объём данных (число объектов, размер файла);
- ограничения, возможные смещения и риски использования;
- рекомендуемое разбиение на обучающую/проверочную/тестовую выборки (или сценарий валидации).
