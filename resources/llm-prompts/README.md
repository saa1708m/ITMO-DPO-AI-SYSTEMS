## Требования к добавлению

Для каждого промпта или сценария использования LLM необходимо:

- указать модель или сервис (например, конкретный поставщик LLM);
- описать назначение промпта (генерация кода, документации, идеи архитектуры и т.п.);
- определить входные данные и ожидаемый формат результата;
- описать ограничения (нельзя использовать для экзамена, закрытых тестов, сдачи решения без переработки);
- указать способ верификации ответа (проверка кода, сравнение с эталонным решением, ревью преподавателя).
