# Банки тестовых заданий

В этом разделе фиксируются банки тестовых заданий (например, для самопроверки или входного/выходного тестирования), которые при необходимости могут дополнять КИМ дисциплины.

## Требования к добавлению

Для каждого банка тестовых заданий рекомендуется:

- указывать перечень тем и модулей, которые покрываются;
- описывать количество заданий и уровни сложности;
- фиксировать правила выборки заданий для одного теста (случайная выборка, тематические блоки);
- указывать наличие ключей/правильных ответов и порядок их обновления;
- отражать правила обновления банка (добавление новых заданий, вывод устаревших).
