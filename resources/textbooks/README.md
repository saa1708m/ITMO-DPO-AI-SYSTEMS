## Требования к добавлению

Для каждого учебника или пособия необходимо:

- указывать полное библиографическое описание (автор(ы), название, издательство, год издания);
- при наличии электронной версии — ссылку на официальный источник (издательство, платформа легального доступа);
- описывать, для каких модулей и КИМ используется учебник или пособие в рамках дисциплины;
- не размещать защищённые авторским правом файлы (сканы, pdf‑копии) в репозитории без разрешения правообладателя или явной лицензии на распространение.
