## Требования к добавлению

Для каждого учебника или учебного пособия необходимо:

- указывать полное библиографическое описание: автор(ы), название, издание, место издания, издательство, год издания, ISBN;
- при наличии электронной версии указывать ссылку только на официальный источник (издательство, библиотечную систему, платформу легального доступа);
- кратко описывать, для каких модулей, КИМ или результатов обучения используется учебник;
- не размещать защищённые авторским правом файлы (сканы, PDF, EPUB и т.п.) без разрешения правообладателя;
- при наличии сопутствующих открытых материалов (например, код к книге) отдельно указывать условия их использования и ссылку на официальный репозиторий.

## Примечание по использованию

Учебники из этого раздела используются как дополнительные источники для самостоятельной подготовки, проектной работы и подготовки к промежуточной аттестации. Основными обязательными материалами дисциплины являются РПД, КИМ, методические указания и ресурсы, перечисленные в других разделах репозитория.
